Welcome to **The Feed** — an open-source content platform powered by Astro 5, Cloudflare Pages, and a Telegram-based ingestion pipeline.

## What You Get

- A fast, SEO-optimized website with SSR
- Content types: articles, videos, audio, galleries, quotes
- Stream: AI-powered link recaps via Telegram bot
- Media CDN via Cloudflare R2

## How It Works

1. Send a URL to your Telegram bot
2. The ingest worker fetches content and generates an AI recap
3. Content is pushed to your GitHub repo as MDX
4. Cloudflare Pages auto-deploys

Start exploring the demo content to see all formats in action.
